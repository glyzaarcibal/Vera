# V.E.R.A. (Voice Emotion Recognition Application)

**V.E.R.A.** is a Capstone Thesis project designed to provide an innovative, AI-driven platform for mental health support. It focuses on individuals aged 13 and above, providing a safe, private, and stigma-free environment.

## 🌟 Project Overview

V.E.R.A. goes beyond traditional text-based mental health tools by integrating advanced **Voice Emotion Recognition**. The system analyzes nuances in the user's voice to understand their emotional state and provides empathetic interactions through an AI Avatar companion.

### Key Features

- 🎤 **Voice Emotion Recognition**: Identifies subtle indicators of emotional distress and mood shifts in real-time.
- 🤖 **AI Avatar Companion**: Provides 24/7 empathetic conversation and guidance.
- 🧘 **Mental Health Activities**: Personalized activities tailored to the user's current emotional state.
- 🔍 **Early Detection**: Facilitates early detection of high-risk cases and connects users to licensed mental health professionals.
- 🛡️ **Stigma-Free Environment**: A secure and private platform for expressing true emotions.

## 🚀 Problem Statement

In the Philippines, many individuals aged 13 and above face mental health challenges like anxiety and depression but remain unheard due to:
- High therapy costs
- Social stigma
- Lack of immediate support systems
- Early emotional distress often goes undetected before escalating into crises.

V.E.R.A. addresses these issues by bridging the gap between self-awareness and professional intervention.

## 📁 Repository Structure

The project is organized into three main components:

- **[frontend/](./frontend)**: A React-based web application built with Vite, providing the user interface and AI interactions.
- **[backend/](./backend)**: A Node.js and Express-based server handling the logic, database management, and AI service integrations.
- **[mobile/](./mobile)**: A mobile application built with React Native and Expo for accessibility on the go.

## 🛠️ Tech Stack

- **Frontend**: React, Vite, CSS (Tailwind and Vanilla)
- **Backend**: Node.js, Express, MongoDB
- **Mobile**: React Native, Expo
- **AI**: Voice Emotion Recognition, Chat AI, AI Avatars (D-ID)
