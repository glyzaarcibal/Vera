# Breathing exercise audio

Place your audio files here for the Take a Breath activity.

**English:**
- get-ready-in.mp3
- inhale.mp3
- exhale.mp3
- hold.mp3
- countdown3.mp3, countdown2.mp3, countdown1.mp3

**Tagalog:**
- humandakasabilang.mp3 (get ready)
- huminga.mp3 (inhale)
- huminganangpalabas.mp3 (exhale)
- Pigil.mp3 (hold)
- countdown3tagalog.mp3, countdown2tagalog.mp3, countdown1tagalog.mp3

If a file is missing, the app will play a short fallback tone.
