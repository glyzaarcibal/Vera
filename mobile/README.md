# V.E.R.A. Mobile

This is the mobile application for **V.E.R.A. (Voice Emotion Recognition Application)**, a Capstone Thesis project.

For a full overview of the project, please see the [main README](../README.md).

## 🚀 Features

- **On-the-go Support**: Mobile-optimized access to V.E.R.A.
- **Voice Recording**: Integrated recording for emotion analysis.
- **AI Chat & Avatar**: Mobile interface for the AI companion.
- **Activity Access**: Participate in mental health activities anywhere.

## 🛠️ Technology Stack

- **Framework**: [React Native](https://reactnative.dev/)
- **Build Tool**: [Expo](https://expo.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) (NativeWind) or Vanilla Styles
- **Navigation**: [React Navigation](https://reactnavigation.org/)

## 📦 Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Set up Environment Variables**:
   Create a `.env` file in this directory with the backend API URL.

3. **Run the application**:
   ```bash
   npx expo start
   ```

4. **Testing on Physical Device**:
   Download the **Expo Go** app and scan the QR code from the terminal.

## 📂 Structure

- `app/Screens`: Application screens and navigation.
- `app/Components`: Mobile-specific UI components.
- `assets/`: App icons, splash screens, and images.
- `utils/`: API handlers and helper functions.
